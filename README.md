# Balolos AI Tools

Affiliate content site — generated and managed by Menace.

## Deployment

1. Push this directory to a GitHub repository
2. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
3. Your site will be live at `https://yourusername.github.io/repo-name`

## Revenue

- Amazon Associates: add your tag to `AMAZON_AFFILIATE_TAG` env var
- SaaS affiliates: apply to programs listed in `AffiliatePublisher_Worker.py`

## Setup Checklist

- [ ] Create GitHub repo
- [ ] Enable GitHub Pages
- [ ] Apply for Amazon Associates (free): https://affiliate-program.amazon.com/
- [ ] Apply for 2-3 SaaS affiliate programs (free — links in bot code)
- [ ] Update `SITE_URL` env var with your real GitHub Pages URL
- [ ] Update `AMAZON_AFFILIATE_TAG` with your real tag
